# BuyMe
Inventory and sales management software
## Technologies
Project is created with:
* Asp.net core mvc: 5
* Sql Server
* EntityFrameworkCore(Code First Approach)
* MediatR
* C#
* Fluent Validation
* AutoMapper
* SyncFussion third-party UI

## Pattern and Architecture
Project is created with:
* CQRS DesignPattern
* Clean Architecture(Onion Architecture)
